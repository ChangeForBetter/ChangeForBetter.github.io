hexo + next 搭建个人博客

发表于 2017-11-27 | 更新于: 2017-12-06 | 分类于 hexo | | 阅读次数:

hexo安装

需要预先安装 git 和 node
node二进制文件下载链接: https://nodejs.org/dist/latest/
node二进制可执行压缩包解压后需要将bin目录全路径加入环境变量PATH中

hexo基本命令

  1. 建站
    hexo init <hexo-folder>
    cd <hexo-folder>
    npm install
  2. 添加标签页/分类页
    hexo new page tags/categories
    默认没有标签页/分类页
    生成 page 后需要配置 md 属性, type: tags/categories
  3. 列出资源文件信息(post/page/tag/category)
    hexo list post
  4. 生成静态文件(文件存放在public目录)
    hexo generate
  5. 启动 hexo
    hexo server -p 80 -s -l

hexo配置(_config.yml)

使用next主题

  1. 下载 next 到 theme 目录下
    cd <hexo-folder>
    git clone https://github.com/iissnan/hexo-theme-next themes/next
  2. 配置 hexo _config.yml
    theme: next # 主题, 默认是landscape

next主题配置(_config.yml)

文章阅读次数统计/文章评论(LeanCloud)

  1. 配置leancloud, 获取 AppID 和 AppKey
  2. LeanCloud 创建 class Counter(文章阅读次数统计), Comment(文章评论)
  3. 配置next_config.yml

文章 Front-matter 配置

文章搜索(Local Search)

  1. 安装hexo-generator-searchdb, 站点根目录执行命令行
    npm install hexo-generator-searchdb --save

  2. 配置 hexo _config.yml

  3. 配置 next _config.yml

其他

hexo 标签数/分类数显示不一致

  1. 停止hexo
  2. hexo clean
  3. 启动hexo

hexo-markdown-emoji渲染

亲测会导致<!-- more-->失效
emoji小抄链接

  1. 卸载hexo-renderer-marked
    npm uninstall hexo-renderer-marked --save
  2. 安装hexo-renderer-markdown-it, markdown-it-emoji
    npm install hexo-renderer-markdown-it --save
    npm install markdown-it-emoji --save
  3. 配置hexo
